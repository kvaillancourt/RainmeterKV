Rainmeter is a desktop customization tool for Windows. For more information and downloads, visit rainmeter.net.

- Add this to the Variables.inc file

//PictureBox Variables
PictureBox=#SkinsPath#\Kara\Resources\Images\PictureBox\Box.png
PictureBoxBGPath=#SkinsPath#\Kara\Resources\Images\PictureBox\Backgrounds

- Extract the PictureBox.rar to the main folder, i.e: where the Kara.ini file is located

- Extract the PictureBox2.rar to the Images folder inside the Resources folder.
